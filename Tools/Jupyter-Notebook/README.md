## 1. 단축키
<p>

특정 셀을 클릭한 뒤 `esc`(혹은 `shift`+`m`)를 눌러 '셀 선택 모드'로 바꾸고 다음과 같이 단축키를 이용하여 원하는 기능을 수행</p>

<p>

**위로 셀 추가**<br>
a
</p>

<p>

**아래로 셀 추가**<br>
a
</p>

<p>

**선택 셀 삭제**<br>
dd
</p>

<p>

**선택 셀 잘라내기**<br>
x
</p>

<p>

**선택 셀 복사**<br>
c
</p>

<p>

**선택 셀과 아래 셀 병합**<br>
shift + m
</p>

<p>

**위로 셀 추가**<br>
a
</p>

<p>

**실행 결과 열기 / 닫기**<br>
o
</p>

<p>

**선택 셀을 Markdown 셀로 변경**<br>
m
</p>

<p>

**선택 셀을 Code 셀로 변경**<br>
y
</p>

**선택 셀의 '코드 입력 모드'로 돌아가기**<br>
enter키
</p>

## 2. 테마 설정
<p>

테마 리스트 보기
```
jt -l
```
</p>

<p>

테마 적용
```
jt -t monokai -N -cellw 80%
```
</p>

<p>

테마 제거
```
jt -r
```
</p>

<p>

코드 블록 라인 번호 보이게 하기<br>
**페이지 상단 → View → Toggle Line Numbers** 체크
</p>
