### 1.2. NVIDIA 그래픽 드라이버 버전 확인
<p>CUDA 설치 시 NVIDIA 그래픽 드라이버도 같이 설치한다. 충돌을 방지하고자 한다면 CUDA를 설치하기 전에 그래픽 드라이버를 삭제한 후 CUDA를 설치하는 것이 좋다.</p>

<p>그래픽 드라이버를 삭제하기 싫다면 본인이 사용해야 할 CUDA 버전과 호환되는 그래픽 드라이버 버전을 확인한다.</p>
