## 1. 영역 선택 및 복사 붙여넣기
<p>

```
vi test.txt
```
위와 같이 파일을 vi 에디터로 열고, 일반 모드에서 영역 선택을 시작하고 싶은 부분에 커서를 위치시킨다.
</p>
<p>

이후 `v`키를 누르면 키보드 방향키를 이용해서 영역을 선택할 수 있다.
</p>

<p>

영역 선택 후 일반 모드에서 `y`키를 누르면 클립보드에 복사된다.
</p>

<p>

클립보드에 복사한 내용을 붙여넣기 하고 싶은 부분에 커서를 두고 `p`키를 누른다.
</p>
