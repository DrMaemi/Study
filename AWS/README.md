## 2. 클라우드의 여섯가지 장점들
<p>

1. 자본 비용을 가변 비용으로 대체
2. 규모의 경제로 얻게 되는 이점
3. 용량 추정 불필요
4. 속도 및 민첩성 개선
5. 중요한 문제에 집중
6. 몇 분 만에 전 세계 배포
</p>

## 3. 데이터베이스
<p>데이터베이스는 크게 관계형, 비관계형으로 나누어진다. 관계형은 MySQL, 비관계형은 MongoDB가 대표적이다.</p>
<p>AWS에서도 관계형, 비관계형 데이터베이스를 제공한다. 관계형으로 RDS, 비관계형으로 DynamoDB가 있다.</p>
