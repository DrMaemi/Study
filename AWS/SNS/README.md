## SNS 사용 사례
<p>

- 어플리케이션 및 시스템 경보
- 푸시 이메일 및 문자 메시지
- 모바일 푸시 알림
</p>

## SNS 특성
<p>

- 하나의 게시된 메시지
- 리콜 옵션 없음
- HTTP/HTTPS 재시도
- 주문 및 전달 보장 X
</p>
