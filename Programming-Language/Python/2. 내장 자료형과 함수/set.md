# Set: 파이썬 집합 자료형

## 1. 특징
- 중복 허용x
- 순서 없음(Unordered)
순서가 없기 때문에 딕셔너리처럼 인덱싱 불가

## 2. 기본 예제
```
>>> s1 = set([1, 2, 3])
>>> s2 = {4, 5, 6}
>>> s1
{1, 2, 3}
>>> type(s2)
<class 'set'>
```
```
>>> s2 = set('Hello')
>>> s2
{'e', 'H', 'l', 'o'}
```

## 3. 교집합
```

>>> s1 = {1, 2, 3, 4, 5, 6}
>>> s2 = set([4, 5, 6, 7, 8, 9])
>>> s1 & s2
{4, 5, 6}
>>> s1.intersection(s2)
{4, 5, 6}
```

## 4. 합집합
```
>>> s1 | s2
{1, 2, 3, 4, 5, 6, 7, 8, 9}
>>> s1.union(s2)
{1, 2, 3, 4, 5, 6, 7, 8, 9}
```

## 5. 차집합
```
>>> s1 - s2
{1, 2, 3}
>>> s2 - s1
{8, 9, 7}
>>> s1.difference(s2)
{1, 2, 3}
>>> s2.difference(s1)
{8, 9, 7}
```

## 6. add
set 자료형에 값 하나를 추가한다.
```
>>> s1 = {1, 2, 3}
>>> s1.add(4)
>>> s1
{1, 2, 3, 4}
```

## 7. update
set 자료형에 값 여러 개를 추가한다.
```
>>> s1 = {1, 2, 3}
>>> s1.update([4, 5, 6])
>>> s1
{1, 2, 3, 4, 5, 6}
```

## 8. remove
특정 값을 제거한다.
```
>>> s1 = {1, 2, 3}
>>> s1.remove(2)
>>> s1
{1, 3}
```
